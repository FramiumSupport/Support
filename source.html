Purpose of Each File

c2me.toml

Controls parallel chunk loading to improve performance in large worlds.

entityculling.properties

Skips rendering of invisible entities, reducing GPU usage.

ferritecore.properties

Optimizes memory by reducing redundant data.

krypton.properties

Improves multiplayer connection by compressing network packets.

lithium.properties

Speeds up game calculations (physics, AI, etc.) for better FPS.

sodium-options.json

Advanced configuration for the graphics renderer.

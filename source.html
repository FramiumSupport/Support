Estructura del Código Fuente

📁 FRAMIUM/
📁 config/
📄 c2me.toml - Configuración para la carga optimizada de chunks
📄 dynamicfps.json - Ajustes para FPS dinámico cuando la ventana está inactiva
📄 entityculling.properties - Optimización de renderizado de entidades
📄 ferritecore.properties - Reducción de uso de memoria RAM
📄 iris.properties - Configuración de shaders (si se usan)
📄 krypton.properties - Optimización de red para multiplayer
📄 lithium.properties - Mejoras generales de rendimiento
📄 sodium-options.json - Ajustes avanzados de renderizado
📁 mods/
📦 (archivos .jar) - Todos los mods de optimización
📄 modlist.json - Lista completa de mods con versiones
🖼️ icon.png - Icono del modpack

Función de cada archivo

c2me.toml

Controla la carga paralela de chunks para mejorar el rendimiento en mundos grandes.

entityculling.properties

Evita renderizar entidades que no son visibles, reduciendo el uso de GPU.

ferritecore.properties

Optimiza el uso de memoria reduciendo datos redundantes.

krypton.properties

Mejora la conexión en multiplayer comprimiendo paquetes de red.

lithium.properties

Acelera cálculos del juego (física, IA, etc.) para mejor FPS.

sodium-options.json

Configuración avanzada del renderizador gráfico.
